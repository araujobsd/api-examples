========================
Gandi API
========================

Introduction
------------------

Gandi採用XML-RPC協定提供遠端的應用程式介面， 讓使用者輕易的打造第三方應用程式並管理您的Gandi服務資源（域名、聯絡人資訊、雲端主機服務等）。此技術文件的完整參考資源為Gandi API v3.0。
相關連結：http://doc.rpc.gandi.net/

API List
------------------
- Gandi Catalog API
- Gandi Certificate API
- Gandi Contact API
- Gandi Domain API
- Gandi Hosting API
- Gandi Notification API
- Gandi Operation API
- Gandi PaaS API
- Gandi Security API
- Gandi Site API
